# ISS Mobile PWA v2

This is a phone only Progressive Web App for ISS monitoring and simple entry workflows.

## Setup

1. Open `config.js`.
2. Put your Supabase Project URL and anon public key inside quotation marks.
3. Upload the folder contents to GitHub Pages.
4. Open the site on a phone in portrait orientation.

Do not put the service role key in `config.js`.

## Main changes in v2

- Top left navigation drawer.
- No bottom navigation buttons.
- Refresh button in the header.
- Logout placed inside the navigation drawer.
- Forgot password changed into a centered link.
- Teacher grade entry now opens a per learner modal.
- Grade correction request and status screens added.
- Advisory class grouped by section cards.
- Adviser and management grade monitoring added.
- Management enrollment summary, classes, search, and PL monitoring simplified for mobile.
